## Description
A simple guessing game, 0 to 100, made in Rust with the help of the 
documentation found in `rustup doc --book` or [clicking here.](https://doc.rust-lang.org/stable/book/)

## How to use
Before cloning this repo make sure you have cargo installed, and then run:
```
git clone https://github.com/GuilhermeDaSilvaVieira/guessing_game.git
```
